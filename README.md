# PinguVPN - IP Addresses

## Правила маршрутизации трафика для Pingu VPN

### 📂 Структура списков

* **`Global`** - перечень IP-адресов и доменов зарубежных сервисов.
* **`RU-RU`** - перечень IP-адресов и доменов сервисов, расположенных в Российской Федерации.

### ⚙️ Конфигурации

* **`doublevpn_ru_bridge_routing`** - конфигурация маршрутизации для сервера-моста.
  Используется для:

  * обхода ограничений белых списков;
  * контроля и ограничения перенаправления трафика на иностранные хостинги.
